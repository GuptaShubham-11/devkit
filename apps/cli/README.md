# 🚀 Devkit CLI

A simple and powerful CLI to create projects from ready-made templates.

---

## ✨ Features

- ⚡ Create projects instantly from templates
- 🔐 Supports authentication (Email / Google)
- 💎 Pro templates with credit system
- 📦 Automatic setup & installation
- 🎯 Clean and interactive terminal experience

---

## 📦 Installation

```bash
npm install -g devkit
```

or

```bash
bun add -g devkit
```

---

## 🚀 Usage

```bash
devkit add <template> <project-name>
```

### Example

```bash
devkit add react-basic my-app
```

```bash
devkit add next-saas-pro my-app
```

---

## 📚 Commands

| Command         | Description                    |
| --------------- | ------------------------------ |
| `devkit add`    | Create a project from template |
| `-h, --help`    | Show help                      |
| `-v, --version` | Show CLI version               |
